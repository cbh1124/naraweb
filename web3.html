Style 태그

style
a {
color:red;
}
/style
색칠한 부위를 선택자라고 부름
a태그를 전부 빨간색으로 바꾼다(효과)

Html이 정보 및 뼈대라면 css는 디자인을 의미한다.

Css의 기본문법

a href="2.html" style="color:red"
Html에서 style라는 속성을 사용하는것이다.

  1. style태그를 쓴다.
  2. style속성을 쓴다.

밑줄을 없애는 style태그를 사용한 코드

style
a {
color:black;
text-decoration: none;
}
/style

특정 열에 밑줄을 넣기 위한 코드

(다음 코드는 style속성을 이용하였다.)

a href="2.html" style="color:red;
text-decoration:underline">CSS
/a

(color:red를 우리는 Declaraion이라고 한다.)
우리는 이를 선언이라고 부른다.

Color:red;에서
color는 property(속성) red는 value(값)
위와 같이 사용할수있다.

css text size property

h1{
font-size:60px;
}
h1태그를 이용한 모든 것들의 size정의한다는 의미

css text center property

h1{
font-size:60px;
text-align: center;
}
